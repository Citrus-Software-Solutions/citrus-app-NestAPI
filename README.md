## Installation & Requirements

- PostgreSQL: 13
- Node.js: 14.17.1
- npm: 6.14.13

```bash
$ npm install
```

## Running the app

```bash
# run server
$ npm run start

# development (watch mode)
$ npm run start:dev

# production
$ npm run start:prod
```

To test the API you must set Environment Variables, follow the .env.sample file to set your own.
