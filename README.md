## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# run server
$ npm run start

# development (watch mode)
$ npm run start:dev

# production
$ npm run start:prod
```

## Test (Not ready yet)
